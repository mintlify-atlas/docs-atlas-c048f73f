The `smartContract()` function provides access to smart contract data deployed on the Hedera network.

## Function signature

```typescript
import { smartContract } from '@tikz/hedera-mirror-node-ts';

const contractCursor = smartContract(client: BaseMirrorClient): SmartContracts
```

<ParamField path="client" type="BaseMirrorClient" required>
  The mirror node client instance used to make API requests
</ParamField>

**Returns:** A `SmartContracts` instance for querying smart contract data

## Basic usage

```typescript
import { Client, smartContract } from '@tikz/hedera-mirror-node-ts';

const client = new Client('https://testnet.mirrornode.hedera.com');

const contractCursor = smartContract(client)
  .setLimit(2)
  .order('asc');

const contracts = await contractCursor.get();
const nextContracts = await contractCursor.next();
```

## SmartContracts class methods

### setSmartContractId()

Filter contracts by contract ID.

```typescript
setSmartContractId(val: OptionalFilters): SmartContracts
```

<ParamField path="val" type="OptionalFilters">
  Contract ID to filter by. Supports comparison operators like `gt:`, `gte:`, `lt:`, `lte:`, `ne:`, or a plain contract ID string
</ParamField>

**Example:**

```typescript
import { optionalFilters } from '@tikz/hedera-mirror-node-ts';

// Get a specific contract
const cursor = smartContract(client)
  .setSmartContractId('0.0.123456');

// Get contracts with ID greater than a value
const cursor2 = smartContract(client)
  .setSmartContractId(optionalFilters.greaterThan('0.0.100000'));
```

### order()

Set the order of results. Inherited from `HasMoreMirrorNode`.

```typescript
order(order: "asc" | "desc"): SmartContracts
```

<ParamField path="order" type="'asc' | 'desc'">
  Sort order for results
</ParamField>

### setLimit()

Set the maximum number of results per page. Inherited from `HasMoreMirrorNode`.

```typescript
setLimit(limit: number): SmartContracts
```

<ParamField path="limit" type="number">
  Maximum number of results to return (default is 25)
</ParamField>

### timestamp()

Filter by timestamp. Inherited from `HasMoreMirrorNode`.

```typescript
timestamp(timestamp: OptionalFilters): SmartContracts
```

<ParamField path="timestamp" type="OptionalFilters">
  Timestamp to filter by. Supports comparison operators
</ParamField>

### get()

Execute the query and return smart contract data.

```typescript
get(): Promise<ContractResponse>
```

**Returns:** A promise that resolves to the contracts response

### next()

Fetch the next page of results.

```typescript
next(): Promise<ContractResponse | null>
```

**Returns:** A promise that resolves to the next page of contracts, or `null` if no more pages exist

## Examples

### Query all contracts

```typescript
import { Client, smartContract } from '@tikz/hedera-mirror-node-ts';

const client = new Client('https://testnet.mirrornode.hedera.com');

const cursor = smartContract(client)
  .setLimit(10)
  .order('desc');

const response = await cursor.get();

console.log(`Found ${response.contracts.length} contracts`);

for (const contract of response.contracts) {
  console.log(`Contract ID: ${contract.contract_id}`);
  console.log(`EVM Address: ${contract.solidity_address}`);
  console.log(`Created: ${contract.created_timestamp}`);
  console.log('---');
}
```

### Filter contracts by ID range

```typescript
import { optionalFilters } from '@tikz/hedera-mirror-node-ts';

const cursor = smartContract(client)
  .setSmartContractId(optionalFilters.greaterThan('0.0.100000'))
  .setLimit(20);

const response = await cursor.get();
```

### Paginate through contracts

```typescript
const cursor = smartContract(client)
  .setLimit(50)
  .order('asc');

let page = 1;
let response = await cursor.get();

while (response) {
  console.log(`Page ${page}: ${response.contracts.length} contracts`);
  
  for (const contract of response.contracts) {
    console.log(`  - ${contract.contract_id}`);
  }
  
  response = await cursor.next();
  page++;
}
```

## Related types

### OptionalFilters

The `OptionalFilters` type allows filtering with comparison operators:

```typescript
type OptionalFilters = 
  | `gt:${string}`    // Greater than
  | `gte:${string}`   // Greater than or equal to
  | `lt:${string}`    // Less than
  | `lte:${string}`   // Less than or equal to
  | `ne:${string}`    // Not equal to
  | `eq:${string}`    // Equal to
  | string;           // Plain value
```

Helper functions are available:

```typescript
import { optionalFilters } from '@tikz/hedera-mirror-node-ts';

optionalFilters.greaterThan(val)
optionalFilters.greaterThanEqualTo(val)
optionalFilters.lessThan(val)
optionalFilters.lessThanEqualTo(val)
optionalFilters.notEqualTo(val)
optionalFilters.equalTo(val)
```
