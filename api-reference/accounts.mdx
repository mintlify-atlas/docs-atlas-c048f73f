title: Accounts
description: Query account information and balances from the Hedera Mirror Node REST API
---

The `accounts()` function provides access to account data, including balances, keys, and transaction history.

## Function signature

```typescript
import { accounts } from '@tikz/hedera-mirror-node-ts';

const accountsCursor = accounts(client: BaseMirrorClient): Accounts
```

<ParamField path="client" type="BaseMirrorClient" required>
  The mirror node client instance used to make API requests
</ParamField>

**Returns:** An `Accounts` instance for querying account data

## Basic usage

```typescript
import { Client, accounts, optionalFilters } from '@tikz/hedera-mirror-node-ts';

const client = new Client('https://testnet.mirrornode.hedera.com');

const accountCursor = accounts(client)
  .setAccountId(optionalFilters.lessThan('0.0.15678177'))
  .setLimit(2);

const accountsData = await accountCursor.get();
const nextAccountsData = await accountCursor.next();
```

## Accounts class methods

### setAccountId()

Filter accounts by account ID with optional comparison operators.

```typescript
setAccountId(val: OptionalFilters): Accounts
```

<ParamField path="val" type="OptionalFilters">
  Account ID to filter by. Supports comparison operators like `gt:`, `gte:`, `lt:`, `lte:`, `ne:`, or a plain account ID string
</ParamField>

**Example:**

```typescript
const cursor = accounts(client)
  .setAccountId('0.0.1234');

// With comparison operators
const cursor2 = accounts(client)
  .setAccountId(optionalFilters.greaterThan('0.0.1000'));
```

### setBalance()

Filter accounts by balance with optional comparison operators.

```typescript
setBalance(val: OptionalFilters): Accounts
```

<ParamField path="val" type="OptionalFilters">
  Balance threshold to filter by. Supports comparison operators
</ParamField>

**Example:**


### get()

Execute the query and return account data.

```typescript
get(): Promise<AccountsResponse>
```

**Returns:** A promise that resolves to the accounts response

### next()

Fetch the next page of results.

```typescript
next(): Promise<AccountsResponse | null>
```

**Returns:** A promise that resolves to the next page of accounts, or `null` if no more pages exist

## Related types

### OptionalFilters

The `OptionalFilters` type allows filtering with comparison operators:

```typescript
type OptionalFilters = 
  | `gt:${string}`    // Greater than
  | `gte:${string}`   // Greater than or equal to
  | `lt:${string}`    // Less than
  | `lte:${string}`   // Less than or equal to
  | `ne:${string}`    // Not equal to
  | `eq:${string}`    // Equal to
  | string;           // Plain value
```

Helper functions are available:

```typescript
import { optionalFilters } from '@tikz/hedera-mirror-node-ts';

optionalFilters.greaterThan(val)
optionalFilters.greaterThanEqualTo(val)
optionalFilters.lessThan(val)
optionalFilters.lessThanEqualTo(val)
optionalFilters.notEqualTo(val)
optionalFilters.equalTo(val)
```
