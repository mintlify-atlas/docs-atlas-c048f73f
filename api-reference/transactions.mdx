The `transactions()` function provides access to transaction data, including transfers, consensus messages, and smart contract calls.

## Function signature

```typescript
import { transactions } from '@tikz/hedera-mirror-node-ts';

const transactionsCursor = transactions(client: BaseMirrorClient, accountId?: string): Transactions
```

<ParamField path="client" type="BaseMirrorClient" required>
  The mirror node client instance used to make API requests
</ParamField>

<ParamField path="accountId" type="string">
  Optional account ID to filter transactions by
</ParamField>

**Returns:** A `Transactions` instance for querying transaction data

## Basic usage

```typescript
import { Client, transactions, TransactionType } from '@tikz/hedera-mirror-node-ts';

const client = new Client('https://testnet.mirrornode.hedera.com');

const transactionCursor = transactions(client)
  .setAccountId('0.0.15678177')
  .setLimit(2)
  .setType('debit')
  .setTransactionType(TransactionType.CONSENSUSSUBMITMESSAGE)
  .setResult('success');

const txns = await transactionCursor.get();
const txns2 = await transactionCursor.next();
```

## Fetch single transaction

```typescript
const txn = await transactions(client)
  .setTransactionId('0.0.1234-1234567890-123456789')
  .get();
```

<Note>
  When a transaction ID is set, all other query parameters are ignored.
</Note>

## Transactions class methods

### setAccountId()

Filter transactions by account ID.

```typescript
setAccountId(val: OptionalFilters): Transactions
```

<ParamField path="val" type="OptionalFilters">
  Account ID to filter by. Supports comparison operators
</ParamField>

**Example:**

```typescript
const cursor = transactions(client)
  .setAccountId('0.0.1234');
```

### setTransactionId()

Fetch a specific transaction by its ID. When set, all other filters are ignored.

```typescript
setTransactionId(val: string | null): Transactions
```

<ParamField path="val" type="string | null">
  Transaction ID to fetch (e.g., "0.0.1234-1234567890-123456789")
</ParamField>

**Example:**

```typescript
const cursor = transactions(client)
  .setTransactionId('0.0.1234-1234567890-123456789');

const txn = await cursor.get();
```

### setResult()

Filter transactions by result status.

```typescript
setResult(val: 'fail' | 'success'): Transactions
```

<ParamField path="val" type="'fail' | 'success'">
  Transaction result to filter by
</ParamField>

**Example:**

```typescript
const cursor = transactions(client)
  .setResult('success');
```

### setType()

Filter by credit or debit transactions.

```typescript
setType(val: 'credit' | 'debit'): Transactions
```

<ParamField path="val" type="'credit' | 'debit'">
  Transaction type - `credit` for incoming, `debit` for outgoing
</ParamField>

**Example:**

```typescript
const cursor = transactions(client)
  .setAccountId('0.0.1234')
  .setType('debit'); // Only outgoing transactions
```

### setTransactionType()

Filter by Hedera transaction type.

```typescript
setTransactionType(val: TransactionType): Transactions
```

<ParamField path="val" type="TransactionType">
  The Hedera transaction type (see [TransactionType enum](#transactiontype-enum))
</ParamField>

**Example:**

```typescript
import { TransactionType } from '@tikz/hedera-mirror-node-ts';

const cursor = transactions(client)
  .setTransactionType(TransactionType.CRYPTOTRANSFER);
```

### order()

Set the order of results. Inherited from `HasMoreMirrorNode`.

```typescript
order(order: "asc" | "desc"): Transactions
```

<ParamField path="order" type="'asc' | 'desc'">
  Sort order for results
</ParamField>

### setLimit()

Set the maximum number of results per page. Inherited from `HasMoreMirrorNode`.

```typescript
setLimit(limit: number): Transactions
```

<ParamField path="limit" type="number">
  Maximum number of results to return
</ParamField>

### timestamp()

Filter by timestamp. Inherited from `HasMoreMirrorNode`.

```typescript
timestamp(timestamp: OptionalFilters): Transactions
```

<ParamField path="timestamp" type="OptionalFilters">
  Timestamp to filter by. Supports comparison operators
</ParamField>

### get()

Execute the query and return transaction data.

```typescript
get(): Promise<TransactionsResponse>
```

**Returns:** A promise that resolves to the transactions response

### next()

Fetch the next page of results.

```typescript
next(): Promise<TransactionsResponse | null>
```

**Returns:** A promise that resolves to the next page of transactions, or `null` if no more pages exist
