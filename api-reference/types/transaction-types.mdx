## Overview

The `TransactionType` enum provides type-safe constants for all transaction types supported by the Hedera network. Use these values to filter transactions by type when querying the Mirror Node API.

## Usage examples

### Filter transactions by type

```typescript
import { RestMirrorNode, TransactionType } from '@hashgraph/hedera-mirror-node';

const mirrorNode = new RestMirrorNode('testnet');

// Get all token creation transactions
const tokenCreations = await mirrorNode.getTransactions({
  transactionType: TransactionType.TOKENCREATION
});

// Get all crypto transfers for an account
const transfers = await mirrorNode.getAccountTransactions('0.0.1234', {
  transactionType: TransactionType.CRYPTOTRANSFER
});
```

### Filter multiple transaction types

```typescript
// Get all token-related transactions
const tokenTxs = await mirrorNode.getTransactions({
  transactionType: [
    TransactionType.TOKENCREATION,
    TransactionType.TOKENMINT,
    TransactionType.TOKENBURN,
    TransactionType.TOKENUPDATE
  ]
});
```

### Filter smart contract transactions

```typescript
// Get all contract calls for a specific contract
const contractCalls = await mirrorNode.getContractTransactions('0.0.5678', {
  transactionType: TransactionType.CONTRACTCALL
});

// Get all contract creation transactions
const contractCreations = await mirrorNode.getTransactions({
  transactionType: TransactionType.CONTRACTCREATEINSTANCE
});
```

### Filter consensus service messages

```typescript
// Get all messages submitted to a topic
const messages = await mirrorNode.getTopicMessages('0.0.9999', {
  transactionType: TransactionType.CONSENSUSSUBMITMESSAGE
});
```

### Check transaction type in results

```typescript
const transactions = await mirrorNode.getTransactions({
  accountId: '0.0.1234',
  limit: 10
});

transactions.transactions.forEach(tx => {
  if (tx.name === TransactionType.CRYPTOTRANSFER) {
    console.log('Found a crypto transfer:', tx.transaction_id);
  } else if (tx.name === TransactionType.TOKENASSOCIATE) {
    console.log('Found a token association:', tx.transaction_id);
  }
});
```

## Notes

- Transaction type values are uppercase strings matching the Hedera protobuf definitions
- Use the enum for type safety and IDE autocomplete support
- Multiple transaction types can be filtered by passing an array
- Not all transaction types may be available on all networks (mainnet, testnet, previewnet)
