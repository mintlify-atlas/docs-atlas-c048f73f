title: Tokens
description: Query fungible token information and balances from the Hedera Mirror Node REST API

The `tokenUtils()` function provides access to fungible token data, including token lists, detailed token information, and token balances.

## Function signature

```typescript
import { tokenUtils } from '@tikz/hedera-mirror-node-ts';

const tokenUtil = tokenUtils(client: BaseMirrorClient): TokenUtils
```

<ParamField path="client" type="BaseMirrorClient" required>
  The mirror node client instance used to make API requests
</ParamField>

**Returns:** A `TokenUtils` instance with access to token-related methods

## Basic usage

```typescript
import { Client, tokenUtils, TokenTypeFilter } from '@tikz/hedera-mirror-node-ts';

const client = new Client('https://testnet.mirrornode.hedera.com');
const tokenUtil = tokenUtils(client);

// Query tokens
const tokenCursor = tokenUtil.Tokens
  .setLimit(2)
  .order('desc')
  .setTokenType(TokenTypeFilter.FUNGIBLE_COMMON);

const tokens = await tokenCursor.get();

// Get token information
const tokenInfo = await tokenUtil.TokenInfo
  .setTokenId('0.0.123456')
  .get();

// Get token balances
const balances = await tokenUtil.TokenBalance
  .setTokenId('0.0.123456')
  .get();
```

## TokenUtils properties

### Tokens

Access the `Tokens` class for querying multiple tokens.

```typescript
tokenUtil.Tokens: Tokens
```

Returns a new `Tokens` instance for querying token data.

**Underlying endpoint:** `GET /api/v1/tokens`

### TokenInfo

Access the `TokenInfo` class for querying detailed information about a specific token.

```typescript
tokenUtil.TokenInfo: TokenInfo
```

Returns a new `TokenInfo` instance for querying detailed token information.

**Underlying endpoint:** `GET /api/v1/tokens/{tokenId}`

### TokenBalance

Access the `TokenBalances` class for querying token balance information.

```typescript
tokenUtil.TokenBalance: TokenBalances
```

Returns a new `TokenBalances` instance for querying token balance data.

**Underlying endpoint:** `GET /api/v1/tokens/{tokenId}/balances`

### setPublicKey()


<ParamField path="val" type="string">

</ParamField>

### setTokenType()

Filter tokens by type (fungible or non-fungible).

```typescript
setTokenType(val: TokenTypeFilter): Tokens
```

<ParamField path="val" type="TokenTypeFilter">
  Token type: `TokenTypeFilter.FUNGIBLE_COMMON` or `TokenTypeFilter.NON_FUNGIBLE_UNIQUE`
</ParamField>

**Example:**

```typescript
import { TokenTypeFilter } from '@tikz/hedera-mirror-node-ts';

const fungibleTokens = await tokenUtil.Tokens
  .setTokenType(TokenTypeFilter.FUNGIBLE_COMMON)
  .get();

const nftTokens = await tokenUtil.Tokens
  .setTokenType(TokenTypeFilter.NON_FUNGIBLE_UNIQUE)
  .get();
```

### TokenBalances

Access token balances for this token without re-specifying the token ID.

```typescript
tokenInfo.TokenBalances: TokenBalances
```

**Example:**

```typescript
const tokenInfo = tokenUtil.TokenInfo.setTokenId('0.0.123456');
const info = await tokenInfo.get();

// Get balances for this token (token ID already set)
const balances = await tokenInfo.TokenBalances.get();
```
