The `nftUtils()` function provides access to NFT (Non-Fungible Token) data, including NFT lists, detailed NFT information, and transaction history.

## Function signature

```typescript
import { nftUtils } from '@tikz/hedera-mirror-node-ts';

const nftUtil = nftUtils(client: BaseMirrorClient): NFTUtils
```

<ParamField path="client" type="BaseMirrorClient" required>
  The mirror node client instance used to make API requests
</ParamField>

**Returns:** An `NFTUtils` instance with access to NFT-related methods

## Basic usage

```typescript
import { Client, nftUtils } from '@tikz/hedera-mirror-node-ts';

const client = new Client('https://testnet.mirrornode.hedera.com');
const nftUtil = nftUtils(client);

// Get NFTs for a token
const nftsCursor = nftUtil.NFTs
  .setTokenId('0.0.123456')
  .order('asc');
const nfts = await nftsCursor.get();

// Get specific NFT info
const nftInfo = await nftUtil.NFTInfo
  .setTokenId('0.0.123456')
  .setSerialNumber(1)
  .get();

// Get NFT transaction history
const nftTxns = await nftUtil.NFTTransactionHistory
  .setTokenId('0.0.123456')
  .setSerialNumber(1)
  .get();
```

## NFTUtils properties

### NFTs

Access the `NFTs` class for querying multiple NFTs.

```typescript
nftUtil.NFTs: NFTs
```

Returns a new `NFTs` instance for querying NFT data.

**Underlying endpoint:** `GET /api/v1/tokens/{tokenId}/nfts`

### NFTInfo

Access the `NFTInfo` class for querying detailed information about a specific NFT.

```typescript
nftUtil.NFTInfo: NFTInfo
```

Returns a new `NFTInfo` instance for querying detailed NFT information.

**Underlying endpoint:** `GET /api/v1/tokens/{tokenId}/nfts/{serialNumber}`

### NFTTransactionHistory

Access the `NFTTransactionHistory` class for querying NFT transaction history.

```typescript
nftUtil.NFTTransactionHistory: NFTTransactionHistory
```

Returns a new `NFTTransactionHistory` instance for querying NFT transaction history.

**Underlying endpoint:** `GET /api/v1/tokens/{tokenId}/nfts/{serialNumber}/transactions`

## NFTs class methods

Query multiple NFTs associated with a token ID or owned by an account.

### setTokenId()

Set the token ID to query NFTs from.

```typescript
setTokenId(val: string): NFTs
```

<ParamField path="val" type="string" required>
  Token ID (e.g., "0.0.123456")
</ParamField>

### setAccountId()

Filter NFTs by owner account ID.

```typescript
setAccountId(val: OptionalFilters): NFTs
```

<ParamField path="val" type="OptionalFilters">
  Account ID to filter by. Supports comparison operators
</ParamField>

**Example:**

```typescript
// Get all NFTs owned by an account
const myNFTs = await nftUtil.NFTs
  .setTokenId('0.0.123456')
  .setAccountId('0.0.1234')
  .get();
```

### get()

Execute the query and return NFT data.

```typescript
get(): Promise<NFTsResponse>
```

### next()

Fetch the next page of results.

```typescript
next(): Promise<NFTsResponse | null>
```

## Advanced examples

### Query NFTs by owner

```typescript
import { Client, nftUtils } from '@tikz/hedera-mirror-node-ts';

const client = new Client('https://testnet.mirrornode.hedera.com');
const nftUtil = nftUtils(client);

// Get all NFTs from a collection owned by a specific account
const cursor = nftUtil.NFTs
  .setTokenId('0.0.123456')
  .setAccountId('0.0.1234')
  .setLimit(50);

const response = await cursor.get();

console.log(`Found ${response.nfts.length} NFTs`);

for (const nft of response.nfts) {
  console.log(`NFT #${nft.serial_number} - Owner: ${nft.account_id}`);
}
```

### Track NFT ownership history

```typescript
const nftUtil = nftUtils(client);

// Get transaction history for a specific NFT
const history = await nftUtil.NFTTransactionHistory
  .setTokenId('0.0.123456')
  .setSerialNumber(42)
  .order('asc')
  .get();

console.log('NFT Ownership History:');
for (const tx of history.transactions) {
  if (tx.type === 'TOKENMINT') {
    console.log(`Minted to ${tx.receiver_account_id} at ${tx.consensus_timestamp}`);
  } else if (tx.type === 'CRYPTOTRANSFER') {
    console.log(`Transferred from ${tx.sender_account_id} to ${tx.receiver_account_id} at ${tx.consensus_timestamp}`);
  }
}
```

### Decode NFT metadata

```typescript
const nft = await nftUtil.NFTInfo
  .setTokenId('0.0.123456')
  .setSerialNumber(1)
  .get();

// Decode base64 metadata
const metadata = Buffer.from(nft.metadata, 'base64').toString('utf-8');
const metadataJson = JSON.parse(metadata);

console.log('NFT Metadata:', metadataJson);
```
